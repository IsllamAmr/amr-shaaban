# نشر المنصة مجانًا على Firebase Spark

هذه النسخة تعمل بدون `Cloud Run` وبدون `Blaze`.

## المطلوب مرة واحدة

1. من Firebase Console افتح `Authentication`.
2. ادخل إلى `Sign-in method`.
3. فعّل `Email/Password`.
4. تأكد أن `Realtime Database` تم إنشاؤها في وضع Locked أو Testing ثم سننشر القواعد من المشروع.

## ملفات النشر المهمة

- `index.html`
- `style.css`
- `script.js`
- `firebase.json`
- `database.rules.json`

## أوامر النشر

```bash
firebase deploy --only hosting,database
```

بعد نجاح النشر سيفتح الموقع على:

```text
https://amr-shaaban.web.app/
```

## أول تشغيل

1. افتح صفحة الموقع.
2. اضغط `دخول المدرس`.
3. إذا لم يكن هناك حساب مدرس بعد، أدخل البريد الإلكتروني وكلمة المرور من نفس الشاشة.
4. سيتم إنشاء حساب المدرس مرة واحدة فقط، ثم يمكنك إنشاء الامتحانات.

## ملاحظة مهمة

هذه النسخة المجانية لا تعرض التصحيح الفوري للطالب بعد التسليم.
السبب أن مفاتيح الإجابة محفوظة في قاعدة البيانات بصلاحيات المدرس فقط حتى لا تصل للطلاب من المتصفح.

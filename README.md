# Amr Shaaban Exam Platform

منصة اختبارات إلكترونية لإدارة الامتحانات ومتابعة الطلاب بشكل بسيط ومنظم.

## فكرة المشروع

المشروع عبارة عن نظام يساعد على:

- إنشاء الاختبارات وإدارتها من لوحة تحكم للأدمن.
- دخول الطلاب إلى الامتحان بالكود الخاص به.
- حفظ المحاولات والنتائج بشكل آمن.
- إدارة بنك أسئلة ورفع مرفقات عند الحاجة.

## المشروع قائم على ماذا؟

- `Node.js` و `Express` لتشغيل الـ backend.
- `Firebase Realtime Database` لحفظ البيانات.
- `Firebase Storage` لحفظ المرفقات.
- `Firebase Hosting` لاستضافة الواجهة الأمامية.
- واجهة أمامية ثابتة داخل `public/`.

## هيكل المشروع الأساسي

- `server.js`: الملف الرئيسي للـ backend.
- `public/index.html`: الصفحة الرئيسية للواجهة.
- `public/style.css`: التنسيق الرئيسي.
- `public/js/`: ملفات JavaScript الخاصة بالواجهة.

## تشغيل المشروع محليًا

```bash
npm install
npm start
```

بعد ذلك افتح:

`http://localhost:3000`

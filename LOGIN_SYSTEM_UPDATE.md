## 🔐 كيفية عمل النظام الجديد

### كلمة المرور الافتراضية:
```

```

### خطوات الدخول:

1️⃣ الدخول إلى **لوحة المدرس**
2️⃣ إدخال **كلمة المرور فقط**

### مخزن الجلسة:

```javascript
localStorage {
```

- صلاحية الجلسة: **24 ساعة**

## 🔧 الدوال الجديدة

### 1. `generateAdminToken(password)`
إنشاء توكن من كلمة المرور
```javascript
const token = generateAdminToken("Amr@2026!");
// يرجع: السلسلة المشفرة
```

### 2. `validateAdminPassword(password)`
التحقق من صحة كلمة المرور
```javascript
if (validateAdminPassword(inputPassword)) {
  // كلمة المرور صحيحة
}
```

### 3. `saveAdminSession(password)`
حفظ الجلسة في localStorage
```javascript
saveAdminSession("Amr@2026!");
// يتم حفظ الجلسة تلقائياً
```

### 4. `getAdminSession()`
الحصول على بيانات الجلسة
```javascript
const session = getAdminSession();
if (session) {
  console.log("الجلسة صالحة");
}
```

### 5. `isAdminSessionValid()`
التحقق من صحة الجلسة
```javascript
if (isAdminSessionValid()) {
  // الجلسة نشطة
}
```

### 6. `clearAdminSession()`
حذف الجلسة
```javascript
clearAdminSession();
// يتم حذف جميع بيانات الجلسة
```

## 🔄 التعديلات على الدوال الموجودة

### `adminLogin()`
✅ الآن تستخدم كلمة المرور فقط
✅ تحفظ الجلسة في localStorage
✅ لا تستخدم Firebase Auth

### `adminLogout()`
✅ حذف الجلسة من localStorage
✅ إعادة التوجيه إلى صفحة الدخول

### `updateAdminLoginView()`
✅ إخفاء حقل البريد الإلكتروني تلقائياً
✅ عرض رسالة موحدة للدخول

### `refreshAdminMode()`
✅ التحقق من الجلسة بدلاً من Firebase
✅ تحديث حالة الدخول

### `ensureAuthReady()`
✅ فحص الجلسة عند التحميل الأول
✅ استعادة جلسة المدرس تلقائياً

### `bootstrap()`
✅ إزالة الاعتماد على Firebase Auth
✅ إضافة فحص الجلسة عند البدء

## ⚠️ ملاحظات مهمة

### 1. تغيير كلمة المرور
لتغيير كلمة المرور:
1. اذهب إلى ملف `.env`
2. غير قيمة `ADMIN_PASSWORD`
3. ابحث عن `validateAdminPassword` في `script.js`
4. أضف الكلمة الجديدة هناك أيضاً

### 2. منتهية الصلاحية
- إذا انتهت صلاحية الجلسة، سيطلب من المدرس الدخول مجدداً
- سيتم حذف البيانات من localStorage تلقائياً

### 3. تسجيل الخروج
- يمكن الخروج من خلال زر الخروج بالصفحة
- أو بمسح بيانات localStorage يدويًا

## 🧪 الاختبار

### تجربة الجلسة الدائمة:
1. ادخل بكلمة المرور ✅
2. ادهب إلى أي صفحة في Admin
3. اضغط F5 (Refresh) 🔄
4. ✅ يجب أن تبقى مسجل دخول

### تجربة انتهاء الصلاحية:
1. ادخل بالمتصفح
2. افتح Console (F12)
3. غير قيمة `admin_expiry` إلى وقت ماضي:
   ```javascript
   localStorage.setItem('admin_expiry', Date.now() - 1000);
   ```
4. اضغط F5 (Refresh)
5. ✅ يجب أن يطلب منك الدخول مجدداً
